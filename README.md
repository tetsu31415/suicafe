## 準備

1. 上記ブログを参考にして各種環境を構築する．
2. `config.py` を修正
3. `se_ok` / `se_bad` ディレクトリ内にSE(.wav)ファイルを入れる．(複数可)
4. 上記ブログを参考にしてDB構築
5. (suicafeをsystemdに追加)
6. (月ごとに `monthly_ranking.py` を実行するようcron登録)
